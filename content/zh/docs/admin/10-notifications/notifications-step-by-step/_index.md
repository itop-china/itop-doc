本章逐步解释如何配置通知，以便在 User Request 分配给代理时发送邮件。通知将发送给工单刚刚分配到的"代理"。

{{% /pageinfo %}}

## 创建触发器

转到"管理工具 / 通知"页面，在"触发器"选项卡中点击"创建新触发器"链接（如果还没有触发器）或点击"新建..."按钮创建新触发器。

当提示选择触发器类型时，选择所需的触发器类型：

点击应用以选择此触发器类型，并在提示时填写请求的字段，其中一些可以是触发器特定的：

然后点击**创建**。

## 创建邮件操作

然后使用面包屑导航返回通知屏幕（您也可以再次使用"通知"菜单）。

在通知屏幕中，跳转到**邮件操作**选项卡：

并点击以创建新的邮件操作。

然后切换到**相关触发器**选项卡，并将此通知链接到您刚刚创建的触发器：

搜索要添加的触发器 → 选择要添加的触发器 → 触发器链接后，点击创建（或应用，如果邮件操作创建已完成）。
