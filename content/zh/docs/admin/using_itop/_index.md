## 连接到 iTop

要访问 iTop，只需将 Web 浏览器指向安装 iTop 的 URL。例如：`http://yourserver/<itop_alias>`

出现提示时，输入您的登录名和密码。

认证后，用户将访问 iTop 主页面。用户首次连接时会显示 "欢迎使用 iTop" 弹出窗口。可以通过取消勾选 "启动时显示此消息" 来在下次不再显示此弹出窗口。

## 在 iTop 中导航

iTop 的主界面如下：

主页面分为三个主要部分：

- **左侧菜单**（也称为资源管理器菜单）包含访问不同模块（CMDB、事件、变更、服务和合同）的链接。显示的列表取决于已安装的模块和当前用户的权限。例如，"管理工具" 菜单（底部）仅对管理员可见。

- **右侧内容面板** 显示搜索结果列表或某个给定项目的详细信息。

- **顶部框架** 包含全局搜索、仪表板菜单（当显示的页面是可编辑仪表板时）以及注销菜单，也用于管理用户偏好设置。

有关如何使用 iTop 各种功能的详细信息，请参阅用户指南。
