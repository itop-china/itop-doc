此菜单提供一些简单的数据库完整性检查。

{{% /pageinfo %}}

## 数据库不一致

提供多种显示模式：

- **错误列表**：仅显示数据库不一致项
- **详细视图**：生成 ID 列表以便生成 SQL SELECT
- **报告**：生成所有不一致项并允许用户将结果下载为 zip 文件

## 丢失附件

您可以在此处搜索数据库中丢失或放错位置的附件。**这不是数据恢复工具，它不会检索已删除的数据。**
