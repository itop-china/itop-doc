## 选择类

右侧浏览器树视图按字母顺序浏览类，支持自动完成过滤。

## 类详细信息选项卡

- **属性**：属性代码、类型、允许值、更多信息、来源
- **相关类**：外键、n-1、1-n、n-n 关系图形视图
- **生命周期**：graphviz 状态转换图，显示事件代码（用于触发器）
- **触发器**：当前类激活的触发器
