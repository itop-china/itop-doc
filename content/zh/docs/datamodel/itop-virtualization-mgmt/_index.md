## 概述

虚拟化管理模块用于管理虚拟化基础设施，包括集群、虚拟机管理程序、逻辑接口和虚拟机。

## 集群

集群（或群集）是一组虚拟机管理程序，它们汇集在一起并共享存储资源，以提供一个整体容错系统来托管虚拟机。

### 集群属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 组织 | 外键指向组织 | 是 |
| 业务关键性 | 可选值：高、低、中 | 否 |
| 投产日期 | 日期（年-月-日） | 否 |
| 描述 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 联系人 | 此配置项的所有联系人 |
| 文档 | 链接到此配置项的所有文档 |
| 工单 | 此配置项的所有工单 |
| 应用解决方案 | 依赖此配置项的所有应用解决方案 |
| 逻辑卷 | 此设备使用的所有逻辑卷 |
| 虚拟机管理程序 | 组成此集群的所有虚拟机管理程序 |
| 虚拟机 | 此主机托管的所有虚拟机 |
| 供应商合同 | 此配置项的所有供应商合同 |

### 创建新集群

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：集群影响其虚拟机、集群所属的应用解决方案以及直接链接到集群的所有联系人（人员、团队）。
- **依赖于**：集群依赖于相关的虚拟机管理程序和逻辑卷。

## 虚拟机管理程序

虚拟机管理程序是一种虚拟化软件（MS Hyper-V、VMWare ESX、Xen 等），运行在物理服务器上并支持创建虚拟机。

### 虚拟机管理程序属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 组织 | 外键指向组织 | 是 |
| 服务器 | 外键指向服务器 | 否 |
| 集群 | 外键指向集群 | 否 |
| 业务关键性 | 可选值：高、低、中 | 否 |
| 投产日期 | 日期（年-月-日） | 否 |
| 描述 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 联系人 | 此配置项的所有联系人 |
| 文档 | 链接到此配置项的所有文档 |
| 工单 | 此配置项的所有工单 |
| 应用解决方案 | 依赖此配置项的所有应用解决方案 |
| 逻辑卷 | 此设备使用的所有逻辑卷 |
| 虚拟机 | 此主机托管的所有虚拟机 |
| 供应商合同 | 此配置项的所有供应商合同 |

### 创建新虚拟机管理程序

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：虚拟机管理程序影响使用它的集群、运行在虚拟机管理程序上的虚拟机、虚拟机管理程序所属的应用解决方案以及直接链接到虚拟机管理程序的所有联系人（人员、团队）。
- **依赖于**：虚拟机管理程序依赖于相关的逻辑卷和托管此虚拟机管理程序的服务器。

## 逻辑接口

虚拟机的网络接口。

### 逻辑接口属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| IP 地址 | IP 地址 | 否 |
| MAC 地址 | 字母数字字符串 | 否 |
| 备注 | 多行字符字符串 | 否 |
| IP 网关 | IP 地址 | 否 |
| IP 掩码 | IP 地址 | 否 |
| 速度 | 十进制值（可为负数） | 否 |
| 虚拟机 | 外键指向虚拟机 | 是 |

## 虚拟机

虚拟机是服务器的虚拟等价物，它托管在虚拟机管理程序或集群上。

### 虚拟机属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 组织 | 外键指向组织 | 是 |
| 状态 | 可选值：实施、已废弃、生产、库存 | 否 |
| 业务关键性 | 可选值：高、低、中 | 否 |
| 虚拟主机 | 外键指向虚拟主机 | 是 |
| 操作系统系列 | 外键指向操作系统系列 | 否 |
| 操作系统版本 | 外键指向操作系统版本 | 否 |
| 操作系统许可证 | 外键指向操作系统许可证 | 否 |
| CPU | 字母数字字符串 | 否 |
| 内存 | 字母数字字符串 | 否 |
| 投产日期 | 日期（年-月-日） | 否 |
| 描述 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 软件 | 此配置项上安装的所有软件 |
| 联系人 | 此配置项的所有联系人 |
| 文档 | 链接到此配置项的所有文档 |
| 工单 | 此配置项的所有工单 |
| 应用解决方案 | 依赖此配置项的所有应用解决方案 |
| 网络接口 | 所有逻辑网络接口 |
| 逻辑卷 | 此设备使用的所有逻辑卷 |
| 供应商合同 | 此配置项的所有供应商合同 |

### 创建新虚拟机

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：虚拟机影响其上运行的软件实例、虚拟机所属的应用解决方案以及直接链接到虚拟机的所有联系人（人员、团队）。
- **依赖于**：虚拟机依赖于相关的逻辑卷以及运行虚拟机的虚拟机管理程序或集群。
