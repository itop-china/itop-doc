## 概述

终端用户设备管理模块用于记录和管理组织中的终端用户设备，包括 IP 电话、手机、PC、外围设备、普通电话、打印机和平板电脑等。

## IP 电话

此对象用于记录直接连接到以太网网络的 IP 电话。

### IP 电话属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 组织 | 外键指向组织 | 是 |
| 状态 | 可选值：实施、已废弃、生产、库存 | 否 |
| 业务关键性 | 可选值：高、低、中 | 否 |
| 位置 | 外键指向位置 | 否 |
| 品牌 | 外键指向品牌 | 否 |
| 型号 | 外键指向型号 | 否 |
| 电话号码 | 字母数字字符串 | 否 |
| 序列号 | 字母数字字符串 | 否 |
| 资产编号 | 字母数字字符串 | 否 |
| 投产日期 | 日期（年-月-日） | 否 |
| 购买日期 | 日期（年-月-日） | 否 |
| 保修结束 | 日期（年-月-日） | 否 |
| 描述 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 联系人 | 此配置项的所有联系人 |
| 文档 | 链接到此配置项的所有文档 |
| 工单 | 此配置项的所有工单 |
| 供应商合同 | 此配置项的所有供应商合同 |

### 创建新 IP 电话

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：IP 电话影响 IP 电话所属的应用解决方案以及直接链接到 IP 电话的所有联系人（人员、团队）。
- **依赖于**：无。IP 电话没有依赖项。
