## 概述

高级存储管理模块用于管理存储基础设施，包括光纤通道接口、逻辑卷、NAS、SAN 交换机、存储系统、磁带和磁带库。

## 光纤通道接口

光纤通道接口是连接到高速网络技术的网络接口，主要用于连接共享存储设备（iTop 中的存储系统）。

### 光纤通道接口属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 设备 | 外键指向数据中心设备 | 是 |
| 速度 | 十进制值（可为负数） | 否 |
| 拓扑 | 字母数字字符串 | 否 |
| WWN | 字母数字字符串 | 否 |

### 创建新光纤通道接口

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

## 逻辑卷

存储系统内部管理的存储单元。

### 逻辑卷属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| LUN ID | 字母数字字符串 | 是 |
| 描述 | 多行字符字符串 | 否 |
| 存储系统 | 外键指向存储系统 | 是 |
| RAID 级别 | 字母数字字符串 | 否 |
| 大小 | 字母数字字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 服务器 | 使用此卷的所有服务器 |
| 虚拟设备 | 使用此卷的所有虚拟设备 |

### 创建新逻辑卷

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：逻辑卷影响使用它的系统（服务器、虚拟机管理程序、集群、虚拟机）。
- **依赖于**：逻辑卷依赖于其存储系统。

## NAS

网络附加存储系统（简称 NAS）是一种连接到网络的专用文件服务器设备。在 iTop 中，NAS 与 NAS 文件系统关联。

### NAS 属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 组织 | 外键指向组织 | 是 |
| 状态 | 可选值：实施、已废弃、生产、库存 | 否 |
| 业务关键性 | 可选值：高、低、中 | 否 |
| 位置 | 外键指向位置 | 否 |
| 机架 | 外键指向机架 | 否 |
| 机箱 | 外键指向机箱 | 否 |
| 品牌 | 外键指向品牌 | 否 |
| 型号 | 外键指向型号 | 否 |
| U 数 | 数值（可为负数） | 否 |
| 序列号 | 字母数字字符串 | 否 |
| 资产编号 | 字母数字字符串 | 否 |
| 投产日期 | 日期（年-月-日） | 否 |
| 购买日期 | 日期（年-月-日） | 否 |
| 保修结束 | 日期（年-月-日） | 否 |
| 电源 A 来源 | 外键指向电源连接 | 否 |
| 电源 B 来源 | 外键指向电源连接 | 否 |
| 描述 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 联系人 | 此配置项的所有联系人 |
| 文档 | 链接到此配置项的所有文档 |
| 工单 | 此配置项的所有工单 |
| 网络接口 | 所有物理网络接口 |
| 光纤通道端口 | 此设备的所有光纤通道接口 |
| 网络设备 | 连接到此设备的所有网络设备 |
| 文件系统 | 此 NAS 中的所有文件系统 |
| 供应商合同 | 此配置项的所有供应商合同 |

### 创建新 NAS

在快速创建区域输入类名以创建新对象，然后填写表单并点击创建按钮。

### 关系

- **影响**：NAS 影响其 NAS 文件系统、NAS 所属的应用解决方案以及直接链接到 NAS 的所有联系人（人员、团队）。
- **依赖于**：NAS 依赖于它连接到的网络设备和 SAN 交换机，以及 NAS 安装的机箱（如果有）。
