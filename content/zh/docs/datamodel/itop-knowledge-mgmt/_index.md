## 概述

已知错误数据库模块用于管理常见问题解答（FAQ）和已知错误，帮助支持代理快速查找解决方案，提高事件处理效率。

## FAQ

常见问题解答（FAQ）回答用户关于服务的最常见问题。FAQ 可以轻松解决常见问题。

### FAQ 属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 类别 | 外键指向 FAQ 类别 | 是 |
| 标题 | 字母数字字符串 | 是 |
| 错误代码 | 字母数字字符串 | 否 |
| 关键词 | 字母数字字符串 | 否 |
| 摘要 | 多行字符字符串 | 否 |
| 描述 | HTML 字符串 | 否 |

## 创建 FAQ

“FAQ”菜单提供 FAQ 列表。只需点击“新建”按钮即可创建新的 FAQ。

## FAQ 类别

对 FAQ 进行分类有助于搜索 FAQ 的代理快速找到所需信息。

### FAQ 类别属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| FAQ | 与此类别相关的所有常见问题解答 |

## 已知错误

已知错误是问题管理流程的一部分。它们用于记录已知问题，即使问题尚未完全修复，例如通过提供变通方法。这提高了事件管理流程的效率。

已知错误可以与配置项关联。当工单与配置项关联时，与该配置项相关的已知错误会自动显示在工单中。

您可以将文档链接到已知错误。

### 已知错误属性

| 名称 | 类型 | 必填？ |
|------|------|--------|
| 名称 | 字母数字字符串 | 是 |
| 客户 | 外键指向组织 | 是 |
| 相关问题 | 外键指向问题 | 否 |
| 错误代码 | 字母数字字符串 | 否 |
| 领域 | 可选值：应用、桌面、网络、服务器 | 是 |
| 供应商 | 字母数字字符串 | 否 |
| 型号 | 字母数字字符串 | 否 |
| 版本 | 字母数字字符串 | 否 |
| 症状 | 多行字符字符串 | 是 |
| 根本原因 | 多行字符字符串 | 否 |
| 变通方法 | 多行字符字符串 | 否 |
| 解决方案 | 多行字符字符串 | 否 |

### 标签页

| 标签页 | 描述 |
|--------|------|
| 配置项 | 与此已知错误相关的所有配置项 |
| 文档 | 链接到此已知错误的所有文档 |

## 创建已知错误

点击“新建已知错误”菜单，将显示以下表单。

除了名称和对应的组织外，已知错误还可以通过以下属性进行描述：

- 错误代码
- 领域
- 供应商
- 型号
- 版本

大多数情况下，已知错误应该与问题关联。这样，支持代理可以快速获取相关信息进行故障排除和修复。
