## 安装您的第一个 iTop

### 下载和安装

iTop 的安装与许多基于 PHP 的应用程序类似：

- 从 Sourceforge 下载 iTop 安装包
- 检查安装先决条件
- 适当调整 PHP 和 MySQL 设置
- 启动安装向导
- 然后通过配置后台任务完成安装

### 实施部署

在您的用户可以开始在生产环境中使用 iTop 之前：

- 阅读用户账户和配置文件，并确定您将使用哪种认证类型
- 阅读实施指南并设置服务目录、客户合同等
- 通过配置电子邮件通知和适当的邮件传输来完成安装
- 调整配置以微调性能
- 做好数据备份的准备
