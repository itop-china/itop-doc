### 日历视图

最后更新于 2026年2月：

- 2.7 分支是第一个长期支持（LTS）版本，已不再维护。
- 3.0 分支已不再维护。
- 3.1 分支已不再维护。
- 3.2 是当前 LTS 版本
- 3.3 分支是下一个短期支持（STS）版本。
- 一个 LTS 持续到下一个 LTS 发布 + 一年的安全补丁
- 一个 STS 持续到下一个版本发布 + 6 个月的安全补丁
- 仅维护扩展的最新版本
