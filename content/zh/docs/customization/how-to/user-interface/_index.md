## 概述

iTop 的用户界面定制涵盖多种常见场景，包括字段控制、显示增强、主题颜色、菜单定制等。本文档汇总了各类界面定制的实现方法。

## 字段控制

- **强制字段必填**：通过多种技术手段强制某个字段必填
- **强制字段只读**：限制特定字段不可编辑
- **对特定用户隐藏字段**：根据用户角色控制字段可见性
- **表单预填充**：自动填充创建表单、搜索表单和转换表单中的字段值
- **数据完整性检查**：在对象保存前验证数据一致性，阻止不合规数据入库
- **定义废弃规则**：设置对象废弃条件
- **防止重复**：通过唯一性规则避免重复数据

## 显示增强

- **修改 iTop 首页**：将控制台首页重定向到任意 iTop 页面
- **在弹出菜单中添加条目**：通过 `iPopupMenuExtension` 接口扩展操作菜单
- **iframe 仪表盘**：创建自定义 iframe 仪表盘组件
- **丰富对象显示**：在对象详情页添加自定义标签页或内容
- **区分同名对象**：通过补充属性区分同名对象
- **修改页面标题**：自定义对象详情页的浏览器标签页标题
- **对象间导航**：在对象详情页添加上一个/下一个导航按钮
- **隐藏影响分析**：移除对象详情页中的特定标签页
- **在列表上添加操作**：为对象列表添加批量操作菜单
- **隐藏对象历史**：对特定用户隐藏活动面板中的历史记录
- **添加对象概览**：定义对象摘要卡片，悬停时显示更多信息
- **排序枚举值**：自定义枚举属性的值排序方式
- **排序转换**：自定义生命周期转换按钮的显示顺序

## 主题与颜色

- **修改 iTop Logo**：替换控制台和登录页面的 Logo
- **修改颜色等**：全面修改 iTop 的颜色方案
- **高亮关键对象**：在列表中突出显示关键对象
- **高亮状态**：为枚举值设置颜色和图标
- **修改菜单图标**：使用 Font Awesome 图标自定义菜单
- **类颜色与图标**：为类设置主色调和图标
- **修改左侧菜单颜色**：自定义左侧导航菜单的颜色
- **修改主色调**：替换 iTop 默认的橙色主色调
- **创建主题**：编写自定义 SCSS 主题
